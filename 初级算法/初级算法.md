### 初级算法

判断质数 90 =  2 * 3 * 3 * 5

```java
class Main {

    public static void main(String[] args) {

        Scanner sc = new Scanner(System.in);
        System.out.println("Input a num:");
        int num = sc.nextInt();
        System.out.println(num + "=");

        for(int i = 2;i <= num;i++) {
            while (num != i) {
                if (num % i == 0) {
                    System.out.print(i + "*");
                    num = num/i;
                }else {
                    break;
                }
            }
        }
        System.out.println(num); // the last num
    }
}		
```
